Джаваскриптоязычный пользовательский сценарий для экспортирования товаров из прогруженной части каталога Ozon в файлы JSON и Markdown.

Сценарий добавляет кнопку «Экспортировать» в правом нижнем углу страницы каталога Ozon. Нажав на неё, вы экспортируете все прогруженные на данный момент товары на странице и они скачаются в виде файлов в форматах JSON и Markdown. В обоих файлах каждый товар имеет название, цену, ссылку на него, рейтинг и количество отзывов.

**Сценарий на стадии альфы и может выдавать ошибки.**

## Установка

1. Установите [расширение Violentmonkey](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/) или аналог.
2. Откройте ссылку на сценарий c [GitHub](https://github.com/RushanM/Ozon-Catalog-Exporter/raw/refs/heads/master/ozon-catalog-exporter.user.js) или [Greasy Fork](https://greasyfork.org/ru/scripts/558509-ozon-catalog-exporter).
3. Установите сценарий.
4. Обновите страницу каталога Ozon, если она была открыта.
5. Готово!

Проект открыт ко вкладам через запросы на слияния (pull requests). Вам могут пригодиться знания GitHub, JavaScript, CSS и HTML. Если запрос на слияние будет неудовлетворительным, об этом будет сообщено.
